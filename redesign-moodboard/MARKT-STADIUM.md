## 1. Waar de markt staat: **Realism (fall)** · 🟢
De AI/automatisering-voor-KMO-markt zit in de **Realism-stage** (de val na de hype). Bewijs uit onze
eigen research:
- **Skepsis is binnen, en verdiend:** koper is *AI-hype-moe, ooit verbrand* (`PDOCS-AVATAR.md` O).
- **Hij wil verifiëren, niet geloven:** *"Is AI automation worth it?" · "31% scared · 41% don't know
  where to start"* (`MARKT-VRAGEN.md` D). Dat is Realism-taal ("is dit echt?"), geen Romance-optimisme.
- **Concurrenten verkopen nog summer:** Krits "AI-first, sneller", de AI-agency-hype (`CONCURRENTIE.md`)
  = infatuation-boodschap aan een fall-markt.

## 2. De timing-these · 🟢
> **Vastpakt is, by instinct, een Realism-operator — en de markt is net in Realism.**
Dat is *"right message, right time"* — de timing waarvan Sultanic zegt dat ze je een genie doet lijken.
Je vecht niet tegen de markt; de stroming draagt je. De concurrenten die nog "AI verandert alles" roepen,
verkopen zomer aan een herfst-markt.

## 3. Realism eist X → Vastpakt doet al Y (de validatie) · 🟢
| Wat Realism eist (Sultanic) | Wat Vastpakt al doet |
|---|---|
| **Currency = waarheid**, niet hype | anti-hype, eerlijk-nee, geen verzonnen claims (STRATEGY §9) |
| **De guru wordt het product** ("wie ben je, ben je een echte operator?") | "praat met wie het bouwt", één zichtbaar mens, `/over-mij/` |
| **Zeg voor wie / niet voor wie** | zelfselectie Wel/Niet, eerlijke diskwalificatie (ADR-0004) |
| **Noem de realiteit, verkoop geen droom** → "finally, someone gets it" | "Je werkt harder dan je bedrijf. Niet jouw schuld." = herkenning |
| **"Waarom werkte de oude manier niet meer"** | het mechanisme: niet jouw schuld, niet een tool — het zit in je hoofd |
| **Done-with-you / nabijheid** (afstand wordt gevaarlijk) | solo done-for-you, "wie bouwt blijft bereikbaar", "erbij blijven" |
| **Integriteit als enige metric** | het hele Honesty Protocol + source-first |
| **"Why you?" (differentiatie luider)** | de lane (eigenaarschap) + "zij verkopen de tool, ik neem het werk weg" |

→ **Je strategie is niet willekeurig "braaf" — ze is stadium-correct.** Dat is de diepere reden dat ze werkt.

## 4. Wat Realism AANSCHERPT (de waarschuwingen) · 🟢
1. **Proof = je #1 hefboom, niet je #1 gap-die-mag-wachten.** In Realism *"the guru becomes the product";
   de koper wil verifiëren dat je een **echte operator** bent. Dun bewijs (1 case, geen quote) is hier de
   gevaarlijkste zwakte. → de #19-quote + 2e case worden **urgent**, niet nice-to-have.
2. **Proof mag niet "too polished" voelen** — Sultanic: de koper *"notices when the proof feels too
   polished."* → een **rauwe** klant-quote verslaat een gladde. Hou het echt.
3. **Nabijheid/continuïteit weegt zwaarder** (de open Q1). "Distance becomes dangerous." → het continuïteit-
   model ("erbij blijven") is in Realism een *verkoop*-argument, geen bijzaak.
4. **De volgende stage is Accountability ("prove it").** Bouw proof **nu** → je loopt vóór op winter.

## 5. Realism-content-taal (het post-template) · 🟢
Sultanic: *"You don't start by asking them to believe a new dream. You start by naming the reality they
are already living in. Then you show them a better way through it."* Het opening-effect = **herkenning**,
niet hype: *"Finally, someone gets it."*

**Post-skelet (Realism, jouw stem):**
1. **Noem de realiteit** die hij privé al voelt (een werkdag-scène uit `MARKT-VRAGEN.md`).
2. **"Waarom de oude manier niet meer werkt"** — het mechanisme (niet jouw schuld; niet een tool).
3. **De eerlijke weg erdoor** (uit je hoofd → systeem dat van jou is), mét de grens (voor wie niet).
4. **CTA = lage drempel** ("Beschrijf één situatie"), geen druk.
