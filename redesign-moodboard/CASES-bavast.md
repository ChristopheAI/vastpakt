# CASE — BAVAST (echt Vastpakt-project)

> Eerste echte case + product-truth, aangeleverd door Christophe (2026-06-03). Vult de 🔴-gaten
> uit COPY-METHOD §5 (product-truth + VoC-echte-klant). Dit is de grondstof voor de scherpste copy
> (Verbist-format: concrete pijn → concreet cijfer → objectie weg). Client anoniem, oplossing genoemd.

## Verbatim (Christophe, ruw)
**Probleem dat BAVAST oploste:** bij de klant zat een belangrijk stuk werk vast in een fragiele
Excel-/Word-flow. Eén iemand wist hoe het bestand werkte, hoe de info geïnterpreteerd moest worden,
waar alles moest komen, welke foto's bij welke ruimte hoorden, hoe het eindrapport opgebouwd moest
worden. Telkens opnieuw: *dictaphone / tekst / foto's / Excel / Word-template / manuele controle /
freelance hulp / wachten / corrigeren.*

> "Het echte probleem was niet 'Excel'. Het probleem was: **de kennis zat in een mens en in een
> gewoonte, niet in een systeem.**" → elk dossier afhankelijk van beschikbaarheid, ervaring,
> concentratie en manueel knip-en-plakwerk.

**Wat BAVAST is (in één zin):** zet ruwe input uit plaatsbeschrijvingen — tekst, structuur en foto's —
automatisch om naar een ingevuld Word-rapport volgens het vaste template van de klant.
Korter: **data erin → BAVAST structureert → rapport eruit.**

**Wat het opleverde (eerste werkende versie):**
- **30 minuten tot 1 uur minder werk per dossier**
- minder manueel knip-en-plakwerk
- minder afhankelijkheid van freelancers
- minder wachttijd tussen opname en rapport
- minder fouten in volgorde, structuur en layout
- consistenter rapport per klant/dossier
- snellere oplevering naar de eindklant

## Waarom deze case de hele positionering bevestigt
- "kennis in een mens, niet in een systeem" = de hero/herkenning-stelling, nu bewezen i.p.v. beweerd.
- De proces-ketting (dicteren → overtikken → knippen → plakken → controleren → wachten) is precies
  het concrete beeld dat de copy miste (Verbist-ijklat: laat het zien, niet de categorie).
- "op het vaste template van de klant" = de brug-belofte: op wat je al gebruikt, niet alles vervangen.
