# Vastpakt Workflows

This folder stores prompts that produced useful, repeatable outputs for the Vastpakt homepage.

Save a prompt here when it creates a reusable workflow, not when it is only a one-off chat instruction.

## Current Workflows

- `2026-05-27-vastpakt-new-session-handoff-prompt.md`  
  Start a new Codex session without losing homepage state, rules, and latest loop context.

- `2026-05-27-vastpakt-whole-homepage-copy-grill.md`  
  Pressure-test the full homepage against docs, source-first rules, and the so-what test.

- `2026-05-27-vastpakt-so-what-copy-pass.md`

- `2026-05-27-vastpakt-external-architecture-transfer.md`  
  Use an external landing-page/post/source as architecture discipline without copying wording, visuals, or category tone.

- `2026-05-27-vastpakt-mobile-visual-copyflow.md`  
  Fix mobile line breaks and visual reading rhythm without changing copy meaning.

- `2026-05-27-vastpakt-weakest-spot-scan.md`  
  Find the weakest homepage section and either weaken, remove, or minimally edit it.

- `2026-05-27-vastpakt-person-block-source-pass.md`

## Review Rule

Before reusing any workflow, check the current loop number, `index.html` script query version, and the latest entries in:

- `docs/copy/2026-05-27-vastpakt-homepage-copy-classification.md`
- `docs/experiments/homepage-experiments.tsv`

If they changed, update the workflow prompt before reuse.
