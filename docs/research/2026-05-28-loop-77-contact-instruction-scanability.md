# Loop 77 - Contact Instruction Scanability

## Question

After reapplying the Vastpakt source-first loop and copy grill, where does the action path still feel heavier than needed?

## Sources checked

- `2026-05-28-web-copy-that-sells-reading.md`
- `2026-05-28-psychological-triggers-workbook-reading.md`
- Current homepage contact exchange
- Copy classification through loop 76

## Source signal

Web copy source note: visitors scan first, and the action path should stay clear.

Psychological triggers source note: the page should reduce the feeling of a preparation-heavy intake without inventing anti-sales slogans.

## Verdict

The contact exchange had the right content, but the `Stuur dit` line carried two jobs in one sentence:

- name possible work objects;
- explain the kind of friction that makes the example useful.

At the final form, that density can make the action feel heavier than needed.

## Change

Split the contact instruction into two short sentences:

1. possible work objects;
2. the friction that makes the example relevant.

## Why this is safe

- Same work objects.
- Same action.
- Same promise.
- No new proof claim.
- No new preparation language.
- No new offer layer.

## Boundary

Do not use this as permission to add slogans such as no project brief, no standard package or no preparation. Keep the public line ordinary and tied to the work object.
