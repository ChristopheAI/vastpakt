# Hugarro Study

Source: https://hugarro.be/?utm_source=placegenie.be

## What Hugarro Says Clearly

- `Hugarro AI Agency`
- `AI-automatisering voor slimme ondernemers in Vlaanderen`
- `Hugarro helpt Vlaamse KMO's slimmer werken met AI-automatisering, websites op maat en digitale tools.`
- `Van workflow-optimalisatie tot webplatformen.`
- `Wij bouwen oplossingen die je bedrijf tijd besparen en laten groeien.`
- `AI-Besparingscalculator`
- `AI-Readiness Scan`
- `Automatisering workflows`
- `Laat je systemen samenwerken en bespaar tijd.`
- `handmatig werk verdwijnt`

## Useful Market Language

- Vlaamse KMO's
- slimmer werken
- repetitieve taken
- systemen samenwerken
- handmatig werk
- bespaar tijd
- digitale tools
- webplatformen op maat
- data-invoer
- email-verwerking
- rapportage

## Useful Copy/Offer Pattern

Hugarro makes the category easy to understand quickly:

```text
AI agency
-> AI automation
-> free calculator/readiness scan
-> expertise blocks
-> cases
-> testimonials
-> FAQ
```

The strongest pattern for Vastpakt is not the AI positioning. It is the easy entry path:

```text
small tool/check
-> quick insight
-> concrete next conversation
```

## What Vastpakt Must Not Copy

- Do not lead with `AI Agency`.
- Do not make the promise `AI kan je X besparen` unless there is measured proof.
- Do not use `AI-readiness` as the first frame.
- Do not use broad growth language such as `laten groeien` as the main promise.
- Do not use `workflow-optimalisatie` as public hero language.
- Do not imitate the decorative startup/SaaS visual style.

## Vastpakt Translation

Hugarro starts from the solution category:

```text
AI-automatisering
```

Vastpakt should start from the work situation:

```text
Waar blijft werk hangen tussen mail, Excel, CRM, facturatie, documenten en mensen?
```

Hugarro offers:

```text
Bereken hoeveel AI kan besparen.
```

Vastpakt should offer:

```text
Toon één voorbeeld waar je vandaag copy-paste, dubbele invoer of vergeten opvolging hebt.
```

## Next Question For Vastpakt

Should Vastpakt have a small entry tool like a check/calculator?

Current answer: maybe, but only if it starts from workday signals, not AI maturity.

Possible direction:

```text
Waar lekt tijd?

Vink aan:
- dezelfde gegevens opnieuw intypen
- offertes opvolgen vanuit mail
- werk klaar maar factuur wacht
- akkoord/bewijs zoeken
- Excel naast CRM blijven bijwerken
```

This can become a low-pressure intake/check, but not a generic calculator.

## Classification

- `Vlaamse KMO's`, `handmatig werk`, `bespaar tijd`, `systemen samenwerken`: market language.
- `AI Agency`, `AI-Readiness`, `AI-Besparingscalculator`: adjacent-category language, useful as contrast.
- `Waar blijft werk hangen tussen mail, Excel, CRM, facturatie, documenten en mensen?`: Vastpakt translation.
- A Vastpakt calculator/check is still a hypothesis until tied to real intake use.
