# Loop 17 - hero workflow without workflow

Date: 2026-05-27

Purpose: bring the loop 16 clarity into the first viewport.

## User Direction

The user confirmed the internal understanding:

```text
Vastpakt bekijkt eigenlijk hoe workflows in bedrijven lopen.
```

## Verdict

Correct internally.

The hero should now say that clearly without using `workflow` as the public frame.

## Source Signals Used

From loop 16:

- `hoe het vandaag loopt`
- `waar info vandaag zit`
- `waar info binnenkomt`
- `wat blijft wachten`
- `wat daarna moet gebeuren`

From local Vastpakt work objects:

- offerte
- akkoord
- planning
- factuurmoment
- overdracht
- mail
- Excel
- CRM
- geheugen

## Previous Hero Lede

```text
Begin met één normale situatie: een offerte, akkoord, planning, factuurmoment of overdracht waar iemand vandaag moet zoeken in mail, Excel, CRM of geheugen.
```

Classification:

- Vastpakt translation.
- Concrete, but it starts from the visitor action rather than the Vastpakt lens.

## New Hero Lede

```text
Vastpakt kijkt hoe het vandaag loopt: waar een offerte, akkoord, planning of factuurmoment binnenkomt, waar info blijft zitten en wat daarna moet gebeuren.
```

Classification:

- light edit / Vastpakt translation.
- Source-near phrase: `hoe het vandaag loopt`.
- Work objects: offerte, akkoord, planning, factuurmoment.
- Workflow idea translated without saying `workflow`.

## Why This Is Better

The hero now answers the reader faster:

```text
Wat doet Vastpakt?
```

Answer:

```text
Vastpakt kijkt hoe werk vandaag loopt.
```

The current H1 can stay:

```text
Alles loopt. Tot iemand vraagt waar het staat.
```

because it shows the recognizable failure moment without becoming process-consulting language.

## Remaining Risk

The line is clearer, but `info blijft zitten` may still be slightly abstract. If the browser review or reader test feels vague, the next loop should test a more object-heavy variant:

```text
Vastpakt kijkt hoe het vandaag loopt: waar een offerte binnenkomt, waar akkoord of bewijs blijft zitten, en wat daarna moet gebeuren.
```
