# Loop 31 - Direct Response Rhythm Pass

## Source

X post by The Boring Marketer:

```text
how to write better copy than 99% of websites on the internet
(give this image to AI for your next ad/landing page/etc)
```

The attached image teaches:

- headline does most of the work;
- problem should be concrete;
- proof should be specific;
- CTA should be clear;
- rhythm matters: short, breathe, land it.

## Verdict

Useful as a copy-editing lens, not as public language.

Vastpakt should not use the image's direct-response jargon: `agitate`, `transform`, financial/emotional chain, or over-forced conversion language.

## Decision

Apply only a rhythm and specificity pass to existing sourced lines.

No new public claim.
