## Verdict

The hero is no longer the main visual bottleneck. The examples section became the heavier path: at the in-app browser width, the practice cards collapsed to one column and the examples section measured about 2887px tall.

Land-book's relevant pattern is compact modular proof: cards stay grouped long enough to scan as a set before the next CTA.

## Borrow

- Keep related proof cards visually grouped on tablet/desktop widths.
- Let the next CTA arrive sooner after the proof.
- Preserve a mobile fallback where one column is genuinely needed.

## Do Not Copy

- Gallery-card spectacle.
- Testimonial/social-proof blocks without permission.
- Glossy case-study treatment.
- Any stronger result or popularity claim.

## Public Change

No public copy changed.

The examples grid now keeps two columns until the narrower `780px` breakpoint. It no longer collapses to one column at `900px`.
