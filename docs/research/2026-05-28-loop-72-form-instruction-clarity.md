# Loop 72 - Form Instruction Clarity

## Question

After reading the direct-response sources, where is the next offer-clarity weakness?

## Sources checked

- Current contact block and form in `index.html`.
- No B.S. Direct Marketing reading note.
- Copy classification around `Stuur één voorbeeld`.
- Loop 8 source-near offerte language.

## Direct-response signal

The Kennedy source is strict about clear instructions: people should know exactly what to do next. The action should also match the threshold of the offer.

For Vastpakt, the first action is low-threshold:

`Stuur één voorbeeld.`

The form should therefore ask for an example, not ask the visitor to diagnose the business problem.

## Verdict

The form label `Waar verlies je vandaag tijd?` was understandable, but slightly too diagnostic.

It asks the visitor to answer the problem frame. The better instruction is closer to the action already repeated across the page: show one example.

## Change

Changed the textarea label to:

`Welk voorbeeld wil je tonen?`

Changed the placeholder from a retyping/facturation example to the source-near offerte follow-up scene:

`Bijvoorbeeld: een offerte is verstuurd, maar ik zie niet wie opvolgt of welke laatste stap genomen is.`

## Why this is safe

- It lowers the form threshold.
- It keeps the action tied to one example.
- It uses an existing source-near offerte scene.
- It does not add a new promise, result, report, audit, score, roadmap, or automation claim.

## Boundary

Do not turn the form into a diagnostic intake, project brief, audit request, preparation checklist, or broad problem analysis.
