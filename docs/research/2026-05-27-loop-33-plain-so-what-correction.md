# Loop 33 - Plain So-What Correction

## Question

After the so-what pass, what is the next optimal correction?

## Browser Observation

Loop 32 made the hero land on:

```text
Dan zie je waar de volgende stap verdwijnt.
```

The line passed the so-what test, but it was slightly more interpretive than the rest of the hero. The page works best when the language stays ordinary and close to the work object.

## Decision

Change the hero line to:

```text
Dan zie je welke stap ontbreekt.
```

## Classification

- `welke stap ontbreekt`: light edit from the existing `volgende stap` / `nog geen spoor` / `wat daarna moet gebeuren` language.
- No new proof or result claim.

## Remaining Risk

The line is safer and plainer, but may be less emotionally sharp than `verdwijnt`. Keep it unless browser or reader feedback says the hero lost too much tension.
