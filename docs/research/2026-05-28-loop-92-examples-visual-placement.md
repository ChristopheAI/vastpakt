# Loop 92 - Examples Visual Placement

## Question

Where should the generated Vastpakt work-object visual go on the homepage?

## Source Used

- Loop 91 anonymous proof boundary.
- Generated image from built-in imagegen.
- Current examples section after loop 90.
- Local imagegen skill guidance: project-bound generated images must be copied into the workspace.

## Decision

Place the visual inside the `Praktijkvoorbeelden` section, between the section heading and the four practice cards.

## Why

This is the safest placement because the image supports the shared pattern across the four examples:

- inventory dashboard;
- mailbox/spreadsheet list;
- Word/report object;
- website/module blocks.

It does not become a fifth case and does not sit near the hero as if it were the main proof object.

## Boundary

The visual is illustrative support, not client proof.

Do not use it as a screenshot, testimonial, named case, or result claim.
