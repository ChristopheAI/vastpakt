# 0003 — Homepage-copy: Problem-Solution-lead met één Groot Idee

- **Status:** Aanvaard
- **Datum:** 2026-06-03

## Context
De prospect is een Vlaamse KMO-zaakvoerder die *problem-aware maar sceptisch* is (AI-hype-moe). De markt
is verzadigd (Schwartz sophistication 3+): "win uren terug", "geen hype", "vaste partner" zijn door
iedereen gezegd (zie CONCURRENTIE.md, COPY-PLAYBOOK §1, INTENT-waarom-inhuren.md §4). Een kale belofte
(Promise-lead) landt niet meer bij deze lezer.

## Beslissing
De homepage opent met een **Problem-Solution-lead**, gebouwd op **één Groot Idee (Rule of One):
schuld-absolutie** — *"Je werkt harder dan je bedrijf. Dat is niet jouw schuld."* We winnen op de
**WIE** (zichtbare solo-mens, "praat met wie het bouwt") en de **HOE** (de brug op je bestaande tools,
jij blijft eigenaar), niet op de **WAT**-claim. Toon: nuchter-warm, geen melodrama, geen verzonnen cijfers.
Eén CTA: "Beschrijf één situatie."

## Gevolgen
- Makkelijker: onderscheidt zich in een verzadigde markt op identiteit + mechanisme (Schwartz 4–5).
- Makkelijker: sluit aan op de sceptische lezer; erkent eerst de pijn vóór de belofte.
- Moeilijker: vraagt discipline (één idee in de hero; de rest is bewijs eronder, geen "tossed salad").
- Vervangt de eerdere Promise-lead ("Minder dubbel werk. Duidelijkere opvolging.", v3.8) als opening.
