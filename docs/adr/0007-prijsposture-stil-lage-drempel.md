# 0007 — Prijsposture: stil + lage drempel (geen bedrag op de site)

- **Status:** Aanvaard
- **Datum:** 2026-06-07

## Context
Issue #18 (prijsmodel) gold als dé go-live-blocker. De koper (STRATEGY §2) is koud, problem-aware en
heeft *nog niet beslist dat hij software wil*. Miran (MARKT-TAAL Bron 04) toont een sterk model —
"vanaf €X" + "een gesprek kost niks" + "je betaalt voor resultaat, niet voor uurtjes" — maar zijn
*bedragen* zijn maatwerk-SaaS en niet over te nemen. Een hard hero-bedrag kan een koude koper vroeg
afschrikken of verkeerd ankeren; totale prijsstilte voedt de "is dit duur / is dit een bureau?"-angst.

## Beslissing
**Prijs-transparantie schaalt met funnel-diepte.** De homepage blijft **stil + lage drempel**: gratis
eerste gesprek + vooraf een vaste afspraak, géén bedrag (blok 8 — al geshipt als eerlijke interim). Een
transparant "vanaf"-bedrag (Miran-model) kan later op een diepere aanpak-/prijs-pagina, en wacht dan op
Christophe's eigen floor-cijfer + model (project/maand). De **kmo-portefeuille** als hefboom wordt
geparkeerd tot registratie/voorwaarden uitgezocht zijn — voorlopig buiten de copy.

Besluit Christophe (spar-sessie 2026-06-07): *stil + lage drempel* boven een zichtbaar "vanaf"-bedrag of
een puur waarde-frame.

## Gevolgen
- **#18 is géén go-live-blocker meer** — de site kan live op de eerlijke interim die al geshipt is.
- Houdt de drempel laag voor de koude funnel-top; sluit aan op de wedge (pijn-first, lage drempel).
- Laat ruimte voor een latere "vanaf"-test dieper in de funnel zonder de homepage te belasten.
- **Geaccepteerd risico:** een koper met prijsangst krijgt nog geen anker — bewust gekozen voor de
  koude funnel-top; op te lossen met een latere diepere prijs-sectie.
