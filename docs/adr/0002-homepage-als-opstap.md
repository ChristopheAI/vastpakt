# 0002 — Homepage is een opstap die naar losse pagina's linkt

- **Status:** Aanvaard
- **Datum:** 2026-06-03

## Context
De hoofdnavigatie telt **Voor wie · Werk · Over mij · Contact**. `Werk` en `Over mij` zijn **eigen
pagina's** (`/werk/`, `/over-mij/`), geen homepage-secties. In een vroege copy-draft werden ze fout
behandeld als volledige blokken óp de homepage, waardoor de homepage alles probeerde te dragen.

## Beslissing
De homepage draagt **de narratieve draad** (hero → herkenning → brug → wig → zelfselectie → CTA) en
**verwijst** naar de losse pagina's. Volledige inhoud hoort thuis op de subpagina:
- echte cases → `/werk/`
- het persoonlijke verhaal + foto → `/over-mij/`
- doelgroep-detail → `/voor-wie/`
Op de homepage staat hooguit een korte teaser met een link ("Bekijk echt werk →").

## Gevolgen
- Makkelijker: de homepage blijft scanbaar en kort; elke pagina heeft één duidelijke taak.
- Makkelijker: cases/verhaal kunnen langer en dieper op hun eigen pagina (lang mag, mits nooit saai).
- Moeilijker: de homepage moet sterk genoeg *teasen* om de klik naar de subpagina te verdienen.
